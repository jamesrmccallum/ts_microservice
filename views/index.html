Timestamp Service

This service will accept a date or timestamp in natural language ( January 1 2018 ) or unix timestamp( 1514764800 ) form and covert it to the other type.

Example usage

Request

GET https://quark-bead.glitch.me/1450137600
GET https://quark-bead.glitch.me/December%2015,%202015

Response

{"unix":1450137600,"natural":"December 15, 2015"}
