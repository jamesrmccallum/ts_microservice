FreeCodeCamp Backend Exercises

1: Timestamp Service (/tsservice)

This service will accept a date or timestamp in natural language ( January 1 2018 ) or unix timestamp( 1514764800 ) form and covert it to the other type.

Example usage

Request
GET https://quark-bead.glitch.me/tsservice/1450137600
GET https://quark-bead.glitch.me/tsservice/December%2015,%202015
Response
{"unix":1450137600,"natural":"December 15, 2015"}

2: Header Viewer (/headers)

Sending a GET to this endpoint will return a few identifying headers from your request.

Example usage

Request
GET https://quark-bead.glitch.me/headers
Response
{"ipaddress":"xx.xxx.xx.xxx","language":"en-US","software":"Macintosh; Intel Mac OS X 10_13_2"}
